📦 Controle de Estoque – DOMMA

Este aplicativo permite gerenciar os itens de estoque das obras da
DOMMA, registrar entradas, baixas e devoluções, e gerar relatórios em
Excel ou PDF.

🚀 Como Executar

1.  Baixe e extraia a pasta enviada (ControleEstoque_v1.0.zip) para o
    seu computador.

2.  Abra a pasta e localize o arquivo:

        controle_estoque.exe

3.  Dê duplo clique para abrir o aplicativo.

    -   Pode demorar alguns segundos na primeira execução (é normal).
    -   Se o Windows mostrar um alerta de segurança, clique em Mais
        informações > Executar assim mesmo.

🛠️ Funcionalidades

-   Cadastro de Itens – Adicione novos itens ao estoque de cada obra.
-   Baixa de Itens – Registre a saída de itens para uso.
-   Devolução – Retorne itens ao estoque quando voltarem do uso.
-   Relatórios – Exporte os movimentos para Excel (.xlsx) ou PDF,
    filtrando por data.
-   Indicadores – Veja o total de itens, itens em uso, disponíveis e em
    manutenção.

🗂️ Estrutura da Pasta

-   controle_estoque.exe → Arquivo principal do aplicativo.
-   controle_estoque.db → Banco de dados local (não apague, contém os
    dados de estoque).
-   domma.ico → Ícone do aplicativo (apenas para referência, já embutido
    no .exe).

📅 Notas Importantes

-   O aplicativo usa o banco de dados local (controle_estoque.db).
-   Se quiser começar com um banco vazio, basta excluir o .db antes de
    abrir o app — ele será recriado automaticamente.
-   Sempre use o formato de data AAAA-MM-DD (ex.: 2025-09-21) ao gerar
    relatórios.
